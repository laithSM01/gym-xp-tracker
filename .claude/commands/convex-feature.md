# convex-feature.md

## Purpose

Build a complete feature in GymXP following the standard pattern:
Convex mutation → composable action → Vue UI

## Instructions

1. Read the existing code in the specified files before writing anything
2. Add the mutation in convex/ with proper auth check and validator types
3. Add the action in the specified composable following existing patterns
4. Add the UI in the specified view following existing styling and patterns
5. Do not touch the schema unless explicitly told to
6. Do not change anything outside the three specified files

## Input required

- Mutation name and file
- Fields and types
- Auth rule (who can call this)
- Composable file to update
- View file to update
- Any extra context
